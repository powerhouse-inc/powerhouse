- `useRelationalDb<Schema>()` — no-argument call shape is correct; destructuring `{ db, isLoading, error }` matches `IRelationalDbState<Schema>` field names ✅
- Quick Start example (lines 103–123) calls `useTypedQuery(driveId, callback)` and `useTypedQuery(driveId, callback, params)` — both match the overloads ✅

- Import path `@powerhousedao/reactor-browser/relational` — confirmed as a named subpath export in `packages/reactor-browser/package.json` ✅
- `QueryCallbackReturnType` — described implicitly via the raw-SQL pattern `{ sql, parameters }` — matches `{ sql: string; parameters?: readonly unknown[] }` in source (`useRelationalQuery.ts:9–12`) ✅
