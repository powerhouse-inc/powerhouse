# Learn the Powerhouse stack, chapter by chapter

A linear course covering the mental model, the architecture, document models,
editors, Connect, Switchboard, and sync. Read a little, try a little, quiz
yourself, and move on.

These lessons are **explanation, not step-by-step build instructions** — the
goal is to understand *what* the agent builds for you and *why* it looks the way
it does. When you want to build by hand, head to
[Build](/academy/MasteryTrack/ManualTodoTutorial/CreateNewPowerhouseProject); to
ship with the agent, start at [Get started](/academy/GetStarted/VetraStudio).
