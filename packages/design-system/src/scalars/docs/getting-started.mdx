# Built-in scalars

All the supported built-in scalars are listed below:

## BooleanField

The BooleanField component provides two distinct user interface elements for handling boolean data: Checkboxes and Toggle switches. These components allow users to make binary choices through familiar and intuitive interaction patterns.
