# Installation

<Source language="bash" code={`
npm install --save @powerhousedao/design-system react-hook-form
`}/>

<Source language="bash" code={`
yarn add @powerhousedao/design-system react-hook-form
`}/>

<Source language="bash" code={`
pnpm add @powerhousedao/design-system react-hook-form
`}/>

> Note: [React Hook Form](https://www.react-hook-form.com/) is a peer dependency to use built-in scalars validation.

# Built-in scalars

## Simple Components

- [BooleanField](?path=/docs/document-engineering-simple-components-boolean-field--readme): Booleans can be rendered as a checkbox field or as a toggle field
  - [CheckboxField](?path=/docs/document-engineering-fragments-checkbox-field--readme)
  - [ToggleField](?path=/docs/document-engineering-fragments-toggle-field--readme)
- [EnumField](?path=/docs/document-engineering-simple-components-enum-field--readme): Enum can be rendered as a radio group or as a select/combobox
  - [RadioGroupField](?path=/docs/document-engineering-fragments-radiogroupfield--readme)
  - [SelectField](?path=/docs/document-engineering-fragments-selectfield--readme)
- [NumberField](?path=/docs/document-engineering-simple-components-number-field--readme)
- [StringField](?path=/docs/document-engineering-simple-components-string-field--readme): Strings can be rendered as single line (inputs) or multiline strings (textarea)
  - [TextField](?path=/docs/document-engineering-fragments-textfield--readme)
  - [TextareaField](?path=/docs/document-engineering-fragments-textareafield--readme)
- [UrlField](?path=/docs/document-engineering-simple-components-url-field--readme)
- [AmountField](?path=/docs/document-engineering-simple-components-amount-field--readme)
- [CountryCodeField](?path=/docs/document-engineering-simple-components-country-code-field--readme)
- [PHIDField](?path=/docs/document-engineering-simple-components-phid-field--readme)
  - [AutocompleteField](?path=/docs/document-engineering-fragments-autocomplete-field--readme)
- [AIDField](?path=/docs/document-engineering-simple-components-aid-field--readme)
- [DatePickerField](?path=/docs/document-engineering-simple-components-date-picker-field--readme)
- [TimePickerField](?path=/docs/document-engineering-simple-components-time-picker-field--readme)
